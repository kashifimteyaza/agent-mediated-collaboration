§ Transparency · AI use statement

How this site was built — and what we owe you to say about it

A workshop on contribution dissolution can hardly afford to be opaque about its own production. So: here is what was made by an agent, what was made by a human, and where the boundary lived.

§ 01 / The build

A worked example of the very problem

This site was built by Hauke Sandhaus (Web Chair) using Claude Code (Anthropic's AI coding assistant, model: Claude Opus 4.6) over the course of an evening, working from the LaTeX source of the workshop proposal. The design direction, page structure, deadlines, and copy decisions were Hauke's. The HTML, CSS, and most prose translation from LaTeX were generated by the agent under his direction.

This is exactly the kind of agent-mediated work the workshop interrogates. We mention it not as a confession but as data — a small worked example of the social conditions under which a "site by Hauke" turns into a site whose every line was passed through an agent.

What the agent did

  • Translated the workshop proposal LaTeX into structured HTML
  • Generated the CSS design system (typography, color, layout)
  • Drafted the page copy, building from the proposal text
  • Assembled the organizer list from the .tex source

What Hauke did

  • All decisions about what the site needs to do and contain
  • Design brief: distinct from the previous workshop, archival/documentary aesthetic
  • Choice to use Fraunces + Inter + JetBrains Mono
  • Reviewed and approved every page before publishing
  • The decision to write this page
We argue in the proposal that disclosure is not enough — that contribution dissolution is a structural problem about the conditions under which witnessed contribution is produced, not just a documentation problem. This page does not solve that. It is itself an instance of it.

Documentation trail

  • Git commit history — every change to this site is traceable in the workshop repo. The initial commit reflects an AI-generated scaffold; subsequent commits reflect human revisions.
  • This page — a plain-language account of the build.
