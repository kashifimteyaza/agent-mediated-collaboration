# Agent-Mediated Collaboration — CSCW 2026 Workshop

**"Nobody Did This": Contribution, Originality, and Accountability in Agent-Mediated Collaboration**

A CSCW 2026 workshop on what happens to attribution, originality, and accountability when LLM agents are embedded inside the collaborative loop.

## Repository contents

- [`website/`](website/) — public workshop website (HTML, CSS, deployable to GitHub Pages)
- `proposal-source.tex` — LaTeX source of the workshop proposal
- `proposal-source.bib` — bibliography for the proposal

## Deploying the website

The `website/` directory is a static site. To deploy on GitHub Pages:

1. Go to **Settings → Pages**
2. Set **Source** to `Deploy from a branch`
3. Set **Branch** to `main` and **Folder** to `/website`
4. (Optional) Add a custom domain via the `CNAME` file in `website/`
